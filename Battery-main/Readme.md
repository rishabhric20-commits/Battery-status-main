# 🔋 Battery Status App

A stylish and responsive **Battery Status Web App** built using **HTML, CSS, and JavaScript** that displays real-time battery information using the **Battery Status API**.

The app shows:

- Current battery percentage
- Charging status
- Remaining battery time
- Low battery warning
- Dynamic battery color indicators
- Charging animation effects
